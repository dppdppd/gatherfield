# Draftyard

A visual organizer for Markdown notes — a local-first editor with linked files
and a clickable graph view.

## Download

Get the latest desktop build from the **[Releases](https://github.com/dppdppd/draftyard/releases)** page:

- **Windows** — portable `.exe` (no installer; just run it)
- **Linux** — `.AppImage` (portable), or `.deb` / `.rpm` packages

## Note

This repository hosts release binaries only. The application source is maintained separately.
